# Estrutura-De-Dados
Exercícios 1-2-3

1) Implemente um programa, baseado no programa
ex0Malloc.c para alocar memória para um vetor. O
número de posições do vetor será indicada via
teclado. Após a atribuição de valores as posições do
vetor, o programa deve imprimir (na tela) estes
valores em ordem inversa da inserção.

2) Altere o programa para armazenar uma estrutura em
um vetor.
Exercícios

3) Altere o programa do exercício I, de forma que
receba números inteiros do usuário indefinidamente.
O programa finaliza quando o usuário entrar com
uma letra.
a) Aloque, inicialmente, memória para 5 inteiros;
b) Caso o usuário entrar com mais inteiros, faça a
realocação, alocando espaço para mais 5 inteiros
e assim sucessivamente;
